# 세 번째 회고

> 1월 19일 ~ 23일 까지의 회고와 느낀점 입니다!

## 느낀점

| 날짜        | 회고 느낀점                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 26.01.19    | 타이포그래피에서 `px`, `rem`, `em`, `vw` 단위의 차이를 배웠고, `clamp(min, preferred, max)`를 사용해 반응형 글자 크기를 안전하게 제한하는 방법을 알게 되었다. 박스 모델에서 content → padding → border → margin 순서와 시계 방향(top → right → bottom → left) 규칙을 익혔고, `margin-left: auto`로 가운데/오른쪽 정렬하는 패턴을 배웠다. `inline` 요소는 width/height가 안 먹어서 `inline-block`이나 `block`으로 변경해야 함을 알게 되었다. list-style 속성들과 position(static, relative, absolute, fixed)의 차이, 그리고 background 관련 속성들을 종합적으로 학습했다.                                                                                                                                                                                                                                                    |
| 26.01.20    | CSS Nesting을 통해 중첩 스타일을 작성하는 방법을 배웠다. `&` 연산자를 사용하여 부모 선택자를 참조할 수 있고, Sass 방식(`&__item`, `&--is-active`)과 CSS 방식(`.menu .menu__item`) 두 가지 중첩 방법이 있음을 알게 되었다. Custom Properties(사용자 정의 속성)를 `:root`에서 정의하고 `var(--변수명)`으로 사용할 수 있으며, 폴백 값을 지정하거나 지역 스코프에서 재정의할 수 있음을 배웠다. Float 속성을 통해 요소를 띄우고 `display: flow-root`로 부모 요소가 float된 자식을 포함하도록 하는 방법, `clear: both`로 float 해제하는 방법, 그리고 `shape-outside: circle(50%)`를 사용해 이미지 주변으로 텍스트가 원형으로 감싸지게 하는 방법을 배웠다.                                                                                                                                                                         |
| 26.01.21    | Media Query를 통해 반응형 웹을 구현하는 방법을 배웠다. `@media screen and (min-width: 500px)` 형식으로 장치를 명시하거나 생략할 수 있고, 최신 문법인 `(width >= 500px)` 또는 `(800px >= width >= 500px)`처럼 범위 지정이 가능함을 알게 되었다. Logical Properties(논리 속성)를 통해 다국어 지원에 유용한 속성들을 배웠다. `margin-block`, `margin-inline`, `inline-size`(width), `block-size`(height), `padding-block-start`, `inset` 등 물리적 방향이 아닌 논리적 방향으로 스타일을 지정하는 방법과 `writing-mode: vertical-lr`을 사용한 세로쓰기 모드를 배웠다. `float: inline-end`처럼 논리 값을 사용할 수 있음도 알게 되었다.                                                                                                                                                                                           |
| 26.01.23    | Flexbox를 심도 있게 배웠다. 플렉스 컨테이너에서 `display: flex`, `flex-direction`(row/column/reverse), `flex-wrap`, `flex-flow` 단축 속성을 사용하는 방법을 익혔다. 메인축 정렬(`justify-content`: start/end/center/space-between/space-around/space-evenly), 교차축 정렬(`align-items`: stretch/start/end/center/baseline), 여러 줄 정렬(`align-content`), 아이템 사이 간격(`gap`)을 설정하는 방법을 배웠다. 플렉스 아이템에서 `flex-grow`, `flex-shrink`, `flex-basis`와 단축 속성 `flex`, `order`로 순서 변경, `align-self`로 개별 아이템 정렬하는 방법도 배웠다. `display: inline-flex`와 `display: flow-root`를 활용한 레이아웃 기법도 익혔다.                                                                                                                                                                         |
| 로그인 과제 | 네이버 로그인 폼을 구현하면서 여러 가지를 배웠다. CSS Custom Properties(`:root`에서 `--color-primary`, `--color-border` 등 정의)를 활용해 색상을 일관성 있게 관리하는 방법을 익혔다. `sr-only` 클래스로 스크린 리더용 레이블을 시각적으로 숨기면서 접근성을 확보하는 기법을 적용했다. 커스텀 체크박스는 `input`을 `opacity: 0`으로 숨기고 `label` 안에 checked/unchecked 상태별 SVG 아이콘을 넣어 `:checked + label` 선택자로 토글하는 방식으로 구현했다. `focus-visible` 의사 클래스를 사용해 키보드 탐색 시에만 아웃라인을 표시하여 마우스 사용자와 키보드 사용자 경험을 분리했다. 반응형 디자인은 `@media (min-width: 768px)`로 모바일에서는 세로 배치, 데스크탑에서는 가로 배치로 변경하고 IP 보안 영역을 표시하도록 구현했다. rem 단위를 일관되게 사용하여 사용자 글꼴 크기 설정에 반응하는 레이아웃을 만들 수 있었다. |

## 한 주를 지내며 생각한 것

- CSS의 다양한 속성들을 배우면서 이전에 비효율적으로 작성했던 코드들을 개선할 수 있게 되었다.
- 특히 Flexbox와 Custom Properties는 앞으로 자주 사용하게 될 것 같다.
- 접근성에 대해 배우면서 시각적인 디자인뿐만 아니라 모든 사용자를 고려해야 함을 느꼈다.
- 로그인 과제를 통해 배운 내용들을 실제로 적용해볼 수 있어서 좋았다.
- 반응형 디자인의 중요성을 알게 되었고, 앞으로는 모바일 환경도 고려해서 개발해야겠다.
