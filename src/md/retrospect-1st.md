# 첫 번째 회고

> 한 주간 멋쟁이 사자처럼 부트캠프에서 학습한 시간을 되돌아보고 회고해본다.
> 이때 회고는 학습한 마크다운 문법을 사용하여 작성한다.

---

## 26.01.05

### 배운 것

- 깃 저장소 초기화 및 브런치, Github의 Repository 사용방법
- 폴더 생성, 파일생성, 파일삭제, 폴더안 파일및 폴더 정보들 확인을 터미널을 통해 배움
- Github remote 명령어를 통한 로컬 폴더와 Repository 연결 및 clone 통한 폴더 복사
- Github branch를 이동하는 방법

### 개선점

- 폴더를 생성하는 방법과 폴더를 만드는 방법을 알지만 finder을 통한 폴더와 파일 만들기가 아닌 터미널을 통해 파일과 폴더를 만드는 습관을 가져야 한다.
- 기존에는 branch를 checkout을 통해 사용했었는데 checkout에 많은 역활이 있음을 배워 switch를 사용해서 branch만 사용할 수 있게 습관을 가져야 한다.

## 26.01.09
